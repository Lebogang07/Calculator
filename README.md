# Calculator

This is a simple GUI-based calculator built using Python and the Tkinter library. 
It performs basic arithmetic operations such as addition, subtraction, multiplication, division and a combination of these operations. 

Features:
Addition, 
Subtraction,
Division, 
Multiplication, 
"AC" button to reset input,
"Backspace" button to delete the last charater entered,
Provides real-time results on the display,
Negate button that changes the sign of the current number 

Technologies Used: Visual Studio, Python 3.9, TKinter(built-in Python Gui Package)

How to Run the Project

1. Install Python (version 3.9 or later recommended)
2. Clone this repository: git clone https://github.com/Lebogang07/Calculator.git
3. Run the Python file:Calculator.py
